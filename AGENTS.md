## Project Overview
This is a TOEIC vocabulary learning app built with React Native and Expo.

## Tech Stack
- React Native
- Expo
- JavaScript (no TypeScript)
- AsyncStorage

## Coding Principles
- Keep code simple and readable
- Prefer minimal and focused changes
- Do not introduce new libraries without explanation

## Restrictions
- Do not convert to TypeScript
- Do not refactor the entire project structure at once
- Do not change navigation structure unless requested

## Development Style
- Solo development project
- Stability over feature expansion
